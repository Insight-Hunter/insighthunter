# InsightHunter

AI-powered Financial Operating System for modern small businesses.

## Tech Stack

- Cloudflare Workers
- D1
- KV
- R2
- Vectorize
- Workers AI
- Astro
- Svelte
- TypeScript
- Turborepo

## Requirements

- Node.js 22+
- pnpm 10+

## Install

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Test

```bash
pnpm test
```

This document describes the InsightHunter design system used across the web app and marketing site.

------------------------------------------------------------
Design Philosophy
------------------------------------------------------------
InsightHunter uses a dark neon aesthetic optimized for clarity, contrast, and emotional impact.
The design system is built for:
- Readability
- Consistency
- Scalability
- Performance
- Zero-framework environments

------------------------------------------------------------

------------------------------------------------------------

Muted Text: #9ca3af

Border Strong: #1f2937  
Success: #22c55e

------------------------------------------------------------
3. Spacing & Layout
------------------------------------------------------------
Max width: 1100px  
Grid: responsive auto-fit grid  
Spacing scale: 4, 8, 12, 16, 20, 24, 32, 48  

Containers:
- .main for page content
- .card for elevated sections
- .grid for metric layouts

------------------------------------------------------------
4. Components
------------------------------------------------------------

Cards:
- Rounded corners
- Subtle border
- Elevated background
- Neon shadow

Buttons:
- Primary: accent background, glow shadow
- Secondary: transparent with border
- Link buttons: accent text

Tables:
- Subtle row hover
- Muted headers
- Clean grid lines

Forms:
- Rounded inputs
- Accent focus ring
- Muted labels

Lists:
- No bullets
- Clean spacing

------------------------------------------------------------
5. Shadows & Effects
------------------------------------------------------------
Soft neon glow:
0 0 20px rgba(76, 111, 255, 0.15)

Strong glow:
0 0 40px rgba(76, 111, 255, 0.25)

------------------------------------------------------------
6. Marketing Components
------------------------------------------------------------
Hero sections:
- Radial gradient background
- Large typography
- Centered layout

Feature cards:
- Elevated
- Icon + title + description

Pricing cards:
- Three-column layout
- Highlighted middle tier

Blog cards:
- Minimalist
- Accent links

FAQ blocks:
- Rounded
- Expandable (optional JS)

------------------------------------------------------------
7. Accessibility Considerations
------------------------------------------------------------
- High contrast text
- Large tap targets
- Semantic HTML
- No color-only indicators
- Keyboard-friendly components

------------------------------------------------------------
Summary
------------------------------------------------------------
The InsightHunter design system provides a cohesive, premium, neon-lit aesthetic that scales across the entire platform while remaining lightweight, accessible, and framework-free.

# InsightHunter Bookkeeping App

A full bookkeeping system built on the InsightHunter platform.

## Features
- Ledger management
- Reconciliation workflows
- Financial reporting
- Multi-client support
- Cloudflare Access authentication
- Durable Object-backed ledger state
- R2-backed report exports

## Tech Stack
- Astro
- React
- TypeScript
- Cloudflare Workers
- Durable Objects
- R2 Storage

## Shared Modules
- @insighthunter/shared-types
- @insighthunter/shared-utils
- @insighthunter/compliance-module (UI consistency)

# InsightHunter Bookkeeping

A professional double-entry bookkeeping system with QuickBooks integration and AI-powered insights, built on Cloudflare Workers and Astro.

## Features

### Core Accounting
-  Full double-entry bookkeeping system
-  Financial reports (Balance Sheet, P&L, Cash Flow)
-  Bank reconciliation wizard
-  Client & vendor management
-  Journal entries with auto-balancing

### QuickBooks Integration
- OAuth 2.0 authentication
- Bi-directional sync
- Import accounts, customers, invoices
- Export transactions to QuickBooks
- Real-time webhook notifications

### Puzzle.io-Inspired Features
- Real-time metrics (MRR, ARR, burn rate, runway)
- Budget tracking with variance analysis
- AI-powered insights and anomaly detection
- Revenue and cash flow forecasting
- Scenario planning
- Tags and automation rules
- Month-end closing workflows
- Comprehensive audit trails

### Advanced Features
- Progressive Web App (PWA)
- Durable Objects for data persistence
- Secure OAuth integration
- Interactive charts and dashboards
- Beautiful UI following Apple HIG
